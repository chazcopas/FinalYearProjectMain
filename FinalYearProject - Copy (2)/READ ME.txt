READ ME - Final Year Project = TimeBenderBattle, Merging time travel with FPS.
The submitted files are the source code for the project completed by Charles Copas - 210369223
The files include the scripts used to determine logic and data stored for the project, as well as files that store the assets used as well as each scene for each level.
To run the code download files and you should find an executable file titled FinalYearProject in the build folder included in this material, unity installed is a requirement to run this.
Alternatively you can load the project using unity and then using the build and run function execute the game and be able to play through it.
